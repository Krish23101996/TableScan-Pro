# TableScan Pro

A single-page web application that extracts tables from uploaded PDF or Excel files and converts them into a downloadable Excel file.

## Features

- Upload PDF and Excel files (.pdf, .xlsx, .xls)
- Automatic table detection and scanning
- Preview detected tables in the browser
- Select and export tables as clean Excel files
- Glassmorphic UI with smooth animations
- Fully responsive design

## Tech Stack

- Vanilla HTML + CSS + JavaScript
- PDF.js (PDF rendering and table detection)
- SheetJS (Excel parsing and export)
